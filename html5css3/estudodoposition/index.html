Position

O valor position:static não se posiciona de forma especial(default se omitido)

position:relative se comporta igual ao position:static, exceto se for adicionado as propriedades (top,right,bottom e let) no estilo do elemento

Este elemento está configurado com position:relative,se este elemento não estivesse configurado desta forma seu elemento filho escaparia e se posicionaria em relação ao elemento pai(neste caso o body)

Este elemento esta configurado como position: absolute e esta posicionado em relação ao seu elemento pai(div id="absolute")gui meu pato e solano pato

Um elemento position:fixed sempre é posicionado em relação ao documento html, ou seja ele sempre ficara posicionado no mesmo lugar
